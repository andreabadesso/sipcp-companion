## 2.3. A Dinâmica dos Sistemas

Vimos que todo sistema abriga em sua estrutura uma vocação relacionada com nossos possíveis propósitos. Isto chamaremos de potencial sistêmico.

Ao arranjo das coisas no interior do sistema, que lhe define o potencial num dado instante, chamaremos de condição.

A máquina, que é, ciberneticamente, um sistema dinamizado, atinge seus propósitos, guiando o sistema de uma condição inicial para uma condição final.

A passagem de uma condição C₁ para uma condição C₂ dá-se o nome de transição.

Quando todas as transições se realizam, sendo atingidos os propósitos do sistema, diz-se que ocorreu uma transformação.
