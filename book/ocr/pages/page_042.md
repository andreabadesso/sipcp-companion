entre si. Interação é, por via de consequência, outra condição.

Coisas que se inter-relacionam e interagem, ou estão integradas, ou acabarão por integrar-se. Integração é, obviamente, a terceira condição. Fácil torna-se inferir que se trata de "conditio sine qua non", isto é, deixando de se verificar uma ou duas das condições, o conjunto não constituirá um sistema.

Estruturalmente os sistemas podem ser físicos (SF) ou de gestão (SG).

Sistema Físico é um conjunto coerente de coisas capaz de, em função de sua estrutura, ser orientado para propósitos.

Sistema de Gestão é um conjunto coerente de coisas adrede estruturado para operar um sistema físico orientando-o para os propósitos que o condicionam.
